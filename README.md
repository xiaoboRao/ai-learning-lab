# AI Learning Lab

一个独立的静态教学网站项目，位于 `/Users/rao/Desktop/opensource/ai-learning-lab`。

## 当前信息架构

站点现在采用三级结构：

1. 首页：只展示教学项目列表
2. 项目页：展示该项目的课程列表与源码入口
3. 课程页：逐节进入具体学习内容

## 入口文件

- `index.html`：项目首页
- `project-rag-api.html`：`rag_api` 项目页
- `lesson.html?id=1`：课程详情页
